过年
2019.02.05 家的回忆

窗外，第一颗烟花已经升空，在蓄积了半夜的新年期待后，霎时都响成一片；
春晚主持人齐声贺岁，新年钟声应声敲响，过年的声音热闹而又嘈杂，恍惚间，新的一年就要来了…

备年夜饭

除夕这天，从早上起床开始便是高兴的，这一天里，所有的正事都是为了准备年夜饭。
早些年家里还养年猪，元旦宰杀的年猪在腌制了个把月后正是上好的食材。外公会搬一把小椅子坐在后门口，开始清洗猪肠子，待到我走过去，便开始向我絮叨：“这东西是顶难洗的，却又是桌上顶好吃的一碗”，说完就会自得的嘿嘿笑起来，满足于自己的逻辑；外婆是主角，大小菜样都是经她的手安排的，从午饭后开始，外婆便会在灶台和煤气灶之间来回忙碌，这个锅里炒两下，那个锅里添点料，锅中的热气飘满厨房，张扬着年夜饭的丰盛；老妈一般会帮忙打下手，洗洗绿叶菜、摆摆熟食盘，看我走过就偶尔夹块肉给我吃；老爸则忙着搞卫生，楼梯上下、墙面桌椅，细细的一一打扫过去。虽然有时我也想帮些忙，但家人之间似乎都习惯了自己的角色，我也就默契的做个“小孩”，吃吃看看便好。

摆酒祭祖

每个地方都有不一样的习俗，在我们这个江浙小镇，备好年夜饭后便是摆酒祭祖。
说是祭祖，其实形式已经简约了不少。围桌倒满一圈小酒盅，各配碗筷，碗中留有浅浅一层米饭，桌边摆正座椅，意在邀前人先入座享用。此时的座椅是千万碰不得的，因为不能扰了前人吃饭。桌上朝向大门的方向会摆上一个香烛台，烛台中间插上三炷香，两边各点一根红烛。然后便是烧纸钱，桌前的地上支起一口大铁锅，点起早已备好的纸钱，意在为前人送去物品和祝福。烧完纸钱大家就在桌前对着烛台站成一排，双手合十，躬身祈祷祝福，期间外公还会去桌上添些酒，意在更尽一杯。
年年都是如此，只是从八仙桌和长凳换成了玻璃圆桌和木椅，怕纸钱落灰又把铁锅移到了门前的空地上去烧，也不再跪着做祈祷，自然地，就慢慢少了些味道，烛火也在玻璃瓷砖的辉映中显得不再明亮。

除夕守岁

酒酣饭饱，一家人便会围坐在客厅，等着八点整开始的春晚；电视的声音会调的稍大些，让原本便热闹的节目更添气氛。
我们南方村落，冬天取暖都会用炭盆，小时候没什么零食，就拿着火钳架在炭盆上烤年糕，一条年糕烤熟后掰着吃，那种满足可以充盈一个晚上；老人们不喜欢吃冷的东西，就会把桔子也塞进炭盆的灰堆里热着，然而我们小孩是不喜欢这样的，觉得热过的桔子软塌塌的不好吃。炭火忽明忽暗，把凑近烤火的我们热的满脸通红，一边翻着年糕，一边看着节目，小时候的快乐便不自觉的在屋里散漫开来。
电视兀自的放着，外婆禁不住睡意总会在沙发上先打起盹来，我们一笑就会把外婆吵醒，然后又笑成一片。外公偶尔会吃些零食，以前基本就是瓜子和花生，年纪大了吃东西就慢，于是就会剥的特别认真，像是我们在做题一样。
临近十二点，外公和老爸会把烟花从屋里搬到门口的空地上，还没等到电视里主持人开始拜年，便略微着急地点着烟花。“嘭”的一声，烟花射入高空，照亮了新年的夜晚。


屋里，春晚的主持人开始拜年，灯光映衬下，这些老面孔如老友般热情的问候着；屋外，烟花爆竹声此起彼伏，“嘭嘭”作响间，五彩的光火绚烂了整个小村庄的夜空。这便是过年，热闹、喜悦、充满希望；眼前的家人，远方的亲朋，在光火和问候中交织在一起，共同结出了一个叫“年”的味道。
