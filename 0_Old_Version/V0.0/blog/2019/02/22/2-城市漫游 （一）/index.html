城市漫游（一）
2019.02.22 城市漫游 那些小店

飘零在外，吃总是一件很难解决的事。上班的时候还有食堂，可一旦到了周末，一日三餐便似乎异常艰难。我不喜外卖，出租屋里的厨房又总归觉得不够自在，所以，一处安静又能果腹的小餐馆便是周末最好的安慰。

笼太仔

这家“笼太仔”，在钱江新城砂之船的最角落里，人少，僻静，深得我心。店铺其实不大，靠墙而建两个开间，大约二三十平，除去占了将近一半的厨房，真正堂食的地方也就三五张桌椅，因为空间不宽敞，收银台直接摆在了门外，然后在过道对面又添上了一排桌椅，这才显得富余了起来。看名字就知道，它家是卖小笼包的，刚去的时候，还只是些包子豆浆之类的，后来生意好起来了，便又多出了许多花样，像是烧麦这些。当然我最喜欢的还是豆腐包，口味一如既往，一笼五个，个大皮薄，加上一碗豆浆，吃的饱饱的。
周末如果不忙，找机会总是会跑去吃一顿，走上约十公里为了一顿包子，看起来傻的要死，但其实只要坐下，平日里的不安和局促就会悄然消失。这里没有人前人后的拥挤，也没有食客站在桌边催促着你的位置，店家顾着自己在厨房忙碌，散落的几个食客互不相扰，我思绪轻盈腹中饱暖，一时也就赖着不想起来。

莲池张

“莲池张”是老字号，说是在当年朱元璋打金华时，每天早晨都要喝一碗莲池张氏煮的牛肉汤，最终攻克了金华府。当然不是说面有多神奇，而是它的味道确实一绝。现在市面上好像也有连锁店，但我去的这一家绝不是连锁，门面逼仄，整个店铺拥挤狭长，桌椅只能靠墙而立，人走进去点完单就要落座，不然第二个人很难进来。店主是一对夫妻，另外有一位应该是他们母亲也常有帮衬。我吃过一次，然后就理解了为什么这家装修简陋的小铺门庭若市的原因。其实他家的面很一般，机器压的那种，但那碗汤头绝对是上品，给的牛肉也很够味，不夸张的说，中午吃一碗，到晚上嘴里还有牛肉味儿。
本来我是绝不可能去这种拥挤的地方的，但实在是抵不过那味道的诱惑，我吃过的牛肉面里，没有第二家比这个正宗了。人多的时候，折叠的桌椅都摆到了街面上，虽然吃相不雅，却依然坐的满满当当，其中还不乏西装领带聊着电话生意的，想来在那一碗汤面前，大家都变得平等而又庸俗了吧～

醉西安

作为一个江南人，从小吃的都是清汤鸡蛋挂面，对面食没有太多的欲望。但这家“醉西安”，着实让我领会了些许西安面食的博大精深。
店铺开在中赢国际南面裙楼，比起上面两家，这里就显得宽敞多了。四方的店面有一个很大的后厨间，师傅戴着高帽在里面忙碌；收银的应该是老板，个高挺拔，发型一丝不苟，有着中年人特有的魅力；打杂的店员时常换着面孔，收拾完了就找个空座坐着休息。推门进去，三列桌椅南北而向，因为座位富余，所以人多的时候也不常坐满。我爱吃它家的油泼面，也是那里的招牌，宽扁的面条搁在碗里，配着些许绿叶青葱，上面是一层油泼辣子，滚烫的香油淋在细碎的辣子上，巨香无比，也是这碗面的灵魂所在。吃的时候一定要拌着，面味葱味辣子味混合在嘴里，简直开出了红白绿的各色花儿。
我不算是个吃货，但对于这种绝妙的味道组合，也不禁被折服吸引，也就时常过去光顾，然而那里毕竟算是个热闹的商圈，因此总是要挑在错开高峰的时候。说起来，宽扁的面条平生未见，在这个本就没有归宿感的城市里应该更显陌生才对，但就是一口之后，那里竟也成了自己常来往返的地方。所以人啊，有时候确实要多去尝试才好，我时常这样对自己说着。
