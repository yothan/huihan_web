未来城市 - 青山周平
2019.02.28 他山之石 建筑设计

开篇写青山是因为着实被他圈了粉，不仅是因为他的作品，更在于他的温和、谦逊，以及周身散发着的朴实；他说话温柔声音不大，话里带些奇怪的口音，演讲时还会紧张的冒汗，从不过分强调自己的观点，总是“还好”“也可以”这样。就是这样一个北漂了十多年的日籍设计师，身在北京胡同却思考着未来城市的样子。

“家”的概念

青山做的居住空间设计，其着眼点在“家”的概念思考上，家是家庭的物理载体，在中国的社会背景下，家庭的概念发生了许多变化，从几世同堂，到一家三口，再到现代城市的租住青年，因此家的形态也发生了变化，从适合多人居住的院落，到三室一厅的商品房，然而适合城市青年住居的物理形态一直没有被明确的提出来。参考现代城市的独身青年比例，青山注意到这是一个被压抑的社会矛盾，而对这个问题的思考也正是在探索未来城市该有的样子。

钱江新城旁有一处建筑名为“垂直院落”，普利兹克奖获得者王澍（shù）设计，通过在上下两层间布置类似院落的挑空空间，在高层住宅中实现了可供几世同堂居住的“院落”方案。


共享空间

以家作为分界线，家以内是私密生活，家以外是公共空间，一般的居住设计倾向于在私密空间内完成尽可能多的功能，而青山的想法是将私密空间做更多的压缩，仅保留诸如睡觉等基础需求，以减轻年轻人的负担；同时重点规划公共空间，用以满足年轻人除睡觉以外的其它需求。结果就是，40平的普通公寓做成了绕屋一周的院落；40户的青年社区做成了下层办公上层居住的共享空间；40层的大楼被分割成不同的功能区域，涵盖了居住、工作、娱乐等功能。这种模式其实很像目前的共享经济，是一种比较高效的资源配置方式，在舒适性和便捷性之间提供了一种适合年轻人的方式。

灵感来自胡同

青山的设计灵感大部分来自于他的胡同生活，北京胡同里的人儿三五成群聚在酒吧、街边、树下的生活方式，正是胡同文化里的一种共享。而对于那些在胡同里不穿衣服的中年男人，青山也有着自己的理解：他们应该是将胡同也作为了自己“家”的一部分，所以在家中是不需要太过正式的，只有走到了胡同外面才是需要穿衣服的。“家”的概念也由此有了延伸，包含了自己的私密空间和共享空间。

后记

青山的设计，有着他的共享哲学，共享就是未来“家”的一部分，因此在私密空间和公共空间的处理上，它们的物理距离会变的更短，以一种融合了的方式互相存在。其实最开始的时候（工业革命以前），工作和家庭就是一体的，人们在家里就可以完成所有事情；工业革命之后出现了大规模的集体化生产，为了提高生产效率而有了工厂，因此工作和家庭就分开了。所以这好像又是一种回归。

最后，单身公寓也好，共享空间也罢，它们都是在解决眼下城市青年租住矛盾的道路上构建着未来城市的形态，是这个时代留给建筑师的问题，青山称之为时代的欲望。作为一个设计者，需要去分析和把握时代的欲望，并通过具象的形式去体现和解决它，最终对社会加以正面的引导，这才是设计者的责任所在。

青山提到，在中国社会，城市中青年群体的居住问题相对比较突出，有很多矛盾和需求得不到解决，作为一个有社会责任感的设计师，应该去关注并思考这些；当然这和所处的社会背景有关，青山自己也调侃，如果身在日本，应该会在做老年人的居住改善问题吧。
