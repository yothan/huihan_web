平凡之光·孤旅
2019.05.26 杂文记事 平凡之光

周末，外面下着大雨，雨水斜进来，阳台上一片湿漉。我坐在落地窗前敲着键盘写字，耳里循环着安静的歌，时不时抬头望一望外面。一只麻雀惊慌的飞来躲雨，落在阳台的铁栏杆上，梳理了被打湿的羽翼，又向外张望着远方。刚要热起来的夏天被一阵雨拖慢了脚步，城市郊区的午后依稀有着慵懒的人群，上海还是透露着那份异地的疏离，我也依然看不清未来的路。也许，等自己的脚步更坚定些，就能继续迈步向前吧。

小隐

行走在人群中，就是行走在普世的价值观之中，很难不被周遭影响。倘若孤身一人，感官更是会被放大，无助悲凉便会密密袭来，这或许就是我不喜人群的原因所在吧。
古人有言，小隐于野，大隐于市，说的应该是不同的心境，也只有那些看破红尘、悟得佛心的人，才能真正做到隐于市而不扰于市吧。如我这般慧根短浅，怕是只得小隐才能安心了。

韬晦

将自己的能力展现和目标相匹配，不露锋芒，韬光养晦。
如果别人不知道你的真实目的，他们就无从得知你的下一步计划。
有时候，什么都不做，恰恰是最难做的事情。

佛心

小时候看西游，看的是大圣通天神威斩妖魔；现在看西游，看的是悟空立地成佛护西经；
也许就是有一个自己，深埋心底，有着世所不知的桀骜。
佛法无边，佛心无我，金箍当头，成一个奇奇怪怪的佛。

车站

来上海面试，离约好的时间早到了一个多小时，也没什么去处，就兀自坐在地铁站里的等候椅上茫然。眼前的地铁一趟接着一趟，车门一开便是一阵拥挤和嘈杂，散去以后又只剩安静和空荡。广播里的女声机械又礼貌的播送，旁边座椅上的人来了又走，身边的一切进行着一场预设的循环，只有我陷在原地，仿佛失去了站起来的力量。
按亮手机屏，看了看时间差不多到了，身体竟不自觉地站了起来。整理了下装束，走出地下，外面是一个更大的循环世界。
