萤石云视频 - 移动应用界面设计

Ezviz Cloud Video App UI

萤石是海康威视旗下子品牌，立足于云视频监控技术，面向社会提供互联网安防服务，萤石云 App 是其移动端应用。

为了摆脱海康工业型产品设计的刻板印象，此次设计，打破了“安防”这一概念的局限，从更具普适意义的“数字生活”角度入手，旨在以“云视频监控”为入口，向用户传达一种可能的未来生活方式。而这也正与萤石未来的战略布局不谋而合。

理念的传达需要载体，我们希望能用一种巧妙的线索去暗示用户，“电视”——成了此次设计的绝佳隐喻，将现实中的视频窗口搬到移动端的虚拟空间中，在交互和视觉上给予用户提示，一个虚拟的“数字家”便完成了。
